# Smartpay — учет поездок сотрудников

Веб-приложение на Google Apps Script + Google Sheets.

## Что умеет
- учет поездок сотрудников Smartpay;
- маршрут из точки А в точку Б с возможностью добавлять новые точки;
- выбор транспорта: личный автомобиль или такси;
- автоматический расчет личного авто:
  - первые 3 км = 10 сомони;
  - после 3 км каждый км = 3 сомони;
  - первые 20 минут бесплатно;
  - после 20 минут каждая минута = 0,10 сомони;
- для такси сумма вводится вручную;
- режим «туда и обратно» автоматически удваивает сумму;
- фильтр по месяцам;
- месячный отчет;
- список поездок;
- удаление записей;
- хранение всех данных в Google Sheets.

## Установка
1. Создайте Google Sheets.
2. Откройте **Extensions → Apps Script**.
3. Создайте файлы:
   - `Code.gs`
   - `index.html`
   - `appsscript.json`
4. Скопируйте код из этого репозитория.
5. В `Code.gs` вставьте ID вашей таблицы:
   ```js
   const SPREADSHEET_ID = 'ID_ВАШЕЙ_ТАБЛИЦЫ';
   ```
6. Запустите функцию `setup()` один раз.
7. Нажмите **Deploy → New deployment → Web app**.
8. Доступ: `Anyone with the link` или по вашей политике доступа.

## Рекомендуемые листы
Функция `setup()` сама создаст:
- `Trips`
- `Points`
- `Employees`
- `Settings`
